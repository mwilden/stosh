About Stockfish

Stockfish is a free chess program for the iPhone, iPod Touch and Mac OS X. The source code can be downloaded from the Stockfish web site.

  • Version: Stockfish for iPhone 2.2, 2010-11-08
  • Author e-mail: tord@glaurungchess.com
  • Web site: http://www.stockfishchess.com
  • Copyright (C) Tord Romstad, Marco Costalba, Joona Kiiski 2009-2010

Stockfish's icon was generously contributed by Klein Maetschke. The "Modern" piece set was designed by Bryan Whitby, and used with his permission.
